# Key Information

- [Ecma International ECMAScript](https://www.ecma-international.org/publications-and-standards/standards/ecma-262/) → implemented as JavaScript in browsers
- **Runtime**: engine + call stack + event loop + async (timers, promises)
- **DOM & BOM**: [document](https://developer.mozilla.org/en-US/docs/Web/API/Document), [window](https://developer.mozilla.org/en-US/docs/Web/API/Window), [element manipulation](https://developer.mozilla.org/en-US/docs/Web/API/Element)
- **Events**: listeners, [mouse](https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent)/[keyboard](https://developer.mozilla.org/en-US/docs/Web/API/KeyboardEvent)/[touch](https://developer.mozilla.org/en-US/docs/Web/API/TouchEvent), [DOMContentLoaded](https://developer.mozilla.org/en-US/docs/Web/API/Document/DOMContentLoaded_event), propagation ([bubbling](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Scripting/Event_bubbling)/capturing)
- **Networking**: [Fetch API](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API) → request/response, JSON, HTTP methods
- **Ecosystem**: libraries & frameworks ([JQuery](https://jquery.com/), [Lodash](https://lodash.com/), [D3.js](https://d3js.org/), [React](https://reactjs.org/), [Angular](https://angular.io/), [Vue.js](https://vuejs.org/))
